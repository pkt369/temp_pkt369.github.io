새로운 프로젝트에 제킨스를 적용하기 위해서 알아보았다.

### 원격서버 배포

이전 포스팅에서 git 을 연결하는것 까지 알아보았다. 이제는 Jenkins를 통해 배포하는 방법을 알아보자.

필자는 gradle 을 사용한 스프링 부트 프로젝트를 연동하였으니 참고 바란다.



```sh
./gradlew build
```
